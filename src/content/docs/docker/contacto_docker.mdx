description: Primer contacto con Docker
---

Para tomar un primer contacto con Docker, nos decargaremos una imagen de Docker desde Docker Hub y la ejecutaremos. Usaremos la imagen oficial de 
MySQL. Debéis iniciar Docker Desktop y desde el buscador superior buscar la imagen oficial de MySQL. Una vez encontrada, pulsamos en el botón "Pull" para  
descargarla. Una vez descargada, podemos ejecutar la imagen. Para ello, pulsamos en el botón "Run" y se abrirá una ventana para configurar la 
ejecución del contenedor.

Es importante que leais la documentación de la imagen oficial de MySQL en [Docker Hub](https://hub.docker.com/) para entender las opciones 
de configuración disponibles.

En este primer contacto deberéis usar la imagen oficial de MySQL y configurarla para acceder al contenedor desde el puerto 3307. Para ello,
como hemos dicho anteriomente, debemos configurar la imagen antes de ejecutarla. Una vez montada correctamente accederemos desde Workbench.
