Una vez que has aprendido los coneceptos básicos de Docker y has usado la imagen Docker de MySQL vas aprender a crear tu propia imagen Docker.
Usaremos el .jar de la aplicación que has creado en la tarea anterior y lo empaquetaremos en una imagen Docker.

## Pasos recomendados para afrontar la tarea

<Steps>
  <ol>
    <li>
      <LinkCard
        title="Video explicativo"
        href="https://openwebinars.net/academia/portada/docker-introduccion/"
        description='Ver sección CREACIÓN DE IMÁGENES DOCKER'
      />
    </li>
    <li>
      Una vez creada la imagen Docker tendrás que comprobar que tu imagen funciona correctamente. Para ello crearás un contenedor con esa imagen y tendrás que usar la consola del contenedor para ejecutar el .jar.
    </li>
    <li>
      Una vez comprobado que el contenedor funciona correctamente, debéis subir la imagen a vuestra cuenta de Docker Hub.
    </li>
  </ol>
</Steps>
