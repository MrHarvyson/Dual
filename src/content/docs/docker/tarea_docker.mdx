Una vez que has aprendido los coneceptos básicos de Docker y has usado la imagen Docker de MySQL vas aprender a crear tu propia imagen Docker.
Usaremos el .jar de la aplicación que has creado en la tarea anterior y lo empaquetaremos en una imagen Docker.

## Pasos recomendados para afrontar la tarea

<Steps>

1. [Video explicativo](https://openwebinars.net/academia/portada/docker-introduccion/) (sección CREACIÓN DE IMÁGENES DOCKER) de como crear un Dockerfile (archivo necesario para crear una imagen
Docker a partir de un .jar)

2. Una vez creada la iamgen Docker tendrás que comprueba que tu iamgen funciona correctamente. Para ellos crearas un contenedor con esa imagen y tendreís
que usar la consola del contenedor para ejecutar el .jar.

3. Una vez comprobado que el contenedor funciona correctamente, debéis subir la imagen a vuestra cuenta de Docker Hub.

</Steps>
