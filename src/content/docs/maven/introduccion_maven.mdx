## ¿Qué es Maven?

Apache Maven es una herramienta de automatización y gestión de proyectos de software, principalmente usada en proyectos Java. 
Fue desarrollada por la Fundación Apache con el objetivo de simplificar la construcción de proyectos (build), el manejo de dependencias y la documentación.

## ¿Para qué sirve Maven?

Maven facilita las siguientes tareas:
- Compilar el código fuente
- Ejecutar pruebas automatizadas
- Empaquetar la aplicación (por ejemplo, en un archivo .jar o .war)
- Gestionar dependencias (librerías externas que necesita tu proyecto)
- Generar documentación del proyecto
- Desplegar artefactos en repositorios remotos

## Características clave

- Archivo de configuración centralizado (pom.xml): En este archivo se declara toda la configuración del proyecto, incluyendo sus dependencias, plugins y versiones.
- Gestión automática de dependencias: Maven descarga e incluye automáticamente las librerías necesarias desde repositorios remotos como Maven Central.
- Convención sobre configuración: Maven promueve una estructura estándar de proyecto, lo que reduce la necesidad de configuraciones personalizadas.
